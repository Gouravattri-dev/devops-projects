# 🐧 Linux DevOps Scripts# 

A collection of Bash scripts for system monitoring and automation.

## 📋 Scripts

### system_info.sh
Displays complete system information including:
- OS and Kernel version
- CPU cores and RAM usage
- Disk usage percentage
- IP Address and Hostname
### cpu_monitor.sh
Monitors current CPU usage and displays real-time statistics.

### disk_monitor.sh
Checks disk space usage and alerts when usage crosses a threshold.

### grep_awk_practice.sh
Demonstrates practical use of grep, awk, and sed for text processing.

### process_monitor.sh
Displays the top 5 CPU-consuming and top 5 memory-consuming processes on the system. Useful for quickly identifying resource-heavy processes during troubleshooting.                                                                                                                          
## 🚀 How to Run

chmod +x script_name.sh
./script_name.sh

# Example:
chmod +x process_monitor.sh
./process_monitor.sh
